# Hypothesis Card Template

Lightweight hypothesis tracking for quick validation cycles.

```markdown
# Hypothesis: {Short Title}

**Status**: {untested | testing | validated | invalidated | pivoted}

## Statement
**We believe** {target users}
**will** {expected behavior}
**because** {rationale}

## Test
**We'll know this is true when** {measurable outcome}
**within** {timeframe}

## Kill Criteria
**If** {condition} **then** we abandon this direction.

## Result
- **Date tested**: {date}
- **Outcome**: {pass | fail | inconclusive}
- **Key metric**: {actual value} vs target {target value}
- **Learning**: {what we learned}
- **Next action**: {what we do now}
```
